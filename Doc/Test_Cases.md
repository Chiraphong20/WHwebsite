# การทดสอบการทำงานของระบบ (Test Cases)

เอกสารนี้รวบรวมรายการทดสอบ (Test Cases) สำหรับตรวจสอบช่องโหว่ ข้อผิดพลาด และประกันคุณภาพของแพลตฟอร์ม (QA) ระดับ Client Side รวบรวมหมวดหมู่ขั้นตอนต่างๆ ในระบบ WongHiran Website 

---

## 1. หมวดหมู่การทดสอบ: การแสดงผลและการเข้าถึงทั่วไป (General UI & Routing)

| รหัสทดสอบ | เงื่อนไขเริ่มต้น/พฤติกรรม (Step) | ผลลัพธ์ที่คาดหวัง (Expected Result) | สถานะ |
| :--- | :--- | :--- | :---: |
| **TC-01-1** | เข้าสู่ระบบ Website หน้า Index (`/`) | หน้าแรกปรากฏขึ้น องค์ประกอบเมนู Banner โหลดเสร็จ ไม่เกิน 2 วินาที | ✅ Pass |
| **TC-01-2** | ควบคุม Navbar เมนู "ติดต่อเรา" (`/contact`) | เปลี่ยน View เป็นหน้า Contact แผนที่และฟอร์มติดต่อจะแสดงขึ้นทันที | ✅ Pass |
| **TC-01-3** | เลื่อนสกอร์เมาส์ลงล่าง และกดปุ่ม Float โทรด่วน (Phone Icon) | หากเป็นมือถือ ระบบจะบังคับให้เปิดแป้นพิมพ์โทรศัพท์กรอกเบอร์ `0935022828` ทันที | ✅ Pass |
| **TC-01-4** | เลื่อนสกอร์เมาส์ลงล่าง และกดปุ่ม Float ไลน์ (LINE Icon) | หากติดตั้งแอป LINE ไว้จะพาเข้าหน้าเพิ่มเพื่อน LINE วงษ์หิรัญ ทันที | ✅ Pass |
| **TC-01-5** | ทดสอบผ่าน Simulator หน้าจอมือถือ (Responsive) iPhone 14 | Navbar หดตัวกลายเป็น Hamburger Menu และ Layout ต้องไม่พังทลายออกนอกจอ (Overflow) | ✅ Pass |

## 2. หมวดหมู่การทดสอบ: แคตตาล็อกและฟิลเตอร์ (Product Page & Catalog)

| รหัสทดสอบ | เงื่อนไขเริ่มต้น/พฤติกรรม (Step) | ผลลัพธ์ที่คาดหวัง (Expected Result) | สถานะ |
| :--- | :--- | :--- | :---: |
| **TC-02-1** | เข้าหน้า `/products` | ข้อมูลสินค้า ProductGrid โผล่ขึ้นมาครบถ้วน | ✅ Pass |
| **TC-02-2** | พิมพ์คำค้นหาใน Search Bar (เช่น `รถบังคับ`) | สินค้าที่ชื่อมีคำว่ารถบังคับจะถูกกรองและแสดงออกมา | ✅ Pass |
| **TC-02-3** | กดเลือก Filter เป็นปุ่มไอคอน "เครื่องเขียน" | ผลลัพธ์สินค้าเหลือเฉพาะสินค้าที่มี `category === 'stationery'` | ✅ Pass |
| **TC-02-4** | พยายามคลิกสินค้าที่หมดแล้วจากระบบจำลอง (ถ้ามี) | ปุ่ม Add to cart เปลี่ยนเป็น "หมด" / หรือคลิกไม่ได้ | ⏳ On-Going |

## 3. หมวดหมู่การทดสอบ: ระบบตำแหน่งภูมิศาสตร์ และภายนอก (Google Maps & External App)

| รหัสทดสอบ | เงื่อนไขเริ่มต้น/พฤติกรรม (Step) | ผลลัพธ์ที่คาดหวัง (Expected Result) | สถานะ |
| :--- | :--- | :--- | :---: |

## 4. หมวดหมู่การทดสอบ: การเพิ่มประสิทธิภาพ SEO (SEO Indexing & Tracking)

*ชุดทดสอบนี้ใช้งานเครื่องมือ View Page Source และ `Site Explorer` Tool*

| รหัสทดสอบ | เงื่อนไขเริ่มต้น/พฤติกรรม (Step) | ผลลัพธ์ที่คาดหวัง (Expected Result) | สถานะ |
| :--- | :--- | :--- | :---: |
| **TC-04-1** | กดตรวจโครงสร้าง HTML Tag บนหน้าแรก | เช็คแล้วพบ `<title>` และ `<meta name="description">` ถูกเขียนไว้สวยงาม | ✅ Pass |
| **TC-04-2** | กดตรวจโครงสร้าง Open Graph | มี `<meta property="og:image"` ชี้ไปที่ไฟล์ Logo.png ของร้านค้า | ✅ Pass |
| **TC-04-3** | อิงตัวอักษร JSON-LD ของ Store (WholesaleStore) | JSON-LD วางอยู่พิกัดบรรทัดบนหัว มีพิกัด Latitude และ Longitude ตรงกับสาขาใหญ่ | ✅ Pass |
| **TC-04-4** | เช็คสคริปต์ Google Analytics | โค้ด Tracking ติดตั้งที่ Head และค่าของ Data Layer ทำงานเวลาเปิดเว็บไซต์ | ✅ Pass |
