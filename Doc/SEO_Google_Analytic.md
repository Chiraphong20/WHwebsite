# การวิเคราะห์และปรับแต่งผลลัพธ์ (SEO & Google Analytics)

เอกสารนี้รวบรวมรายละเอียดของโครงสร้าง Search Engine Optimization และการติดตามสถิติผู้เข้าชมเว็บไซต์ **WongHiran Website** ซึ่งการติดตั้งการทำงานอยู่ในรูปแบบเชิงเทคนิคแล้วเสร็จบนไฟล์ `index.html`

---

## 1. การทำเทคนิค SEO บนตัวเว็บไซต์ (On-Page SEO Strategy)

เนื่องจากโครงการนี้เน้นพัฒนาให้เป็นช่องทางการตลาดเบื้องหน้า จึงจำเป็นต้องปูพื้นฐานการดักลีดและการถูกค้นพบให้มีประสิทธิภาพที่สุดผ่าน Metadata Tag

### 1.1 Primary Meta Tags
คือข้อมูลที่แสดงตรงเป็นหัวข้อบนหน้ากระดาน Search Results ของ Google 

| ประเภท | โค้ดที่ครอบไว้ | ผลลัพธ์คาดเกณฑ์และคีย์เวิร์ด |
| :--- | :--- | :--- |
| **Title** | `<title>` | ร้าน 20 บาท, วงษ์หิรัญ ค้าส่ง โคราช, สินค้าเบ็ดเตล็ด |
| **Description**| `<meta name="description">` | แจกแจงว่าที่นี่มีสินค้ามากกว่า 1,000 รายการ และสามารถโทรมาได้เลยเพื่อเร่งยอดคลิก (CTR) |
| **Keywords** | `<meta name="keywords">` | เจาะจงลงพื้นที่ เช่น ค้าส่ง โคราช, ร้านขายส่ง นครราชสีมา (ปทุมธานี/กรุงเทพไม่ต้องเน้น) |

### 1.2 Open Graph Protocol (OG) & Twitter Cards
เพื่อเสริมสร้างเสน่ห์ระดับโซเชียลมีเดียเมื่อมีคนส่งลิงก์ `https://wonghiran.com` ไปให้ในห้องแชทของ **LINE** หรือโพสต์ลงบน **Facebook** องค์ประกอบกราฟิกจะดึง Metadata ด่างๆ ดังนี้:
- **`og:image`** ดึงรูปภาพแบนเนอร์/โลโก้ไปประกอบหน้าพรีวิวให้ลูกค้าเห็น
- **`og:title`** ตัวพิมพ์ใหญ่สุดของหัวข้อลิงก์
- **`og:description`** บรรยายใต้ลิงก์ให้เตะตา

## 2. โครงสร้างโครงข้อมูลพิเศษ (Structured Data / JSON-LD)

**JSON-LD** เป็นภาษาโครงสร้างข้อมูลเบื้องหลังที่เสิร์ชเอนจินอย่าง Google, Bing คุยกันได้โดยไร้อุปสรรค การเขียน Schema ช่วยให้ Google เข้าใจตัวตนของเว็บไซต์นั้นลึกซึ้งยิ่งขึ้น ไม่ใช่แค่มองเป็น HTML ว่างเปล่า

**ประเภทโครงสร้างที่เซ็ตอยู่ในระบบ:** `WholesaleStore` (ธุรกิจค้าส่ง) 

| โครงสร้างที่ส่ง | ตัวแปร/สัดส่วน |
| :--- | :--- |
| **ตำแหน่ง (Address)** | ส่งพิกัดและไปรษณีย์ 30000 เพื่อโชว์ในช่องค้นหาภูมิภาคแบบ Local Search |
| **พิกัด Lat, Lng** | 14.9799, 102.0978 เชื่อมโยงกับ Google Map |
| **เวลาทำการ** | จันทร์-อาทิตย์ 08:00 - 17:00 |
| **ช่องทางอื่น** | เชื่อม Facebook ของวงษ์หิรัญโยงไว้เป็นเครือข่าย `sameAs` เพื่อส่งกำลังของ Backlink ให้กัน |

## 3. ระบบติดตามวิเคราะห์ (Google Analytics 4 Tracking)

เว็บไซต์เชื่อมต่อการดักจับพฤติกรรมคนเข้าเว็ปเพื่อนำข้อมูลไปปิดการขายหรือสร้างแผน Retarget Ads ผ่านชุดข้อมูล gtag.js รุ่นใหม่สุดของฝั่งกูเกิล (GA4)

| ข้อมูล | การเชื่อมต่อ / รหัส |
| :--- | :--- |
| **Measurement ID** | `G-XYDQXXS1P9` |
| **การติดตั้ง** | ผังตัวสคริปต์อยู่ภายในแท็ก `<head>` ของโครงสร้างไฟล์เว็บ |
