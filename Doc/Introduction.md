# บทนำและวัตถุประสงค์ (Introduction and Objectives)

เอกสารฉบับนี้จัดทำขึ้นเพื่ออธิบายความเป็นมา วัตถุประสงค์ และขอบเขตการทำงานของเว็บไซต์หลักทางการ (Official Website) สำหรับร้าน **WongHiran Wholesale (วงษ์หิรัญค้าส่ง โคราช)** ซึ่งเป็นการนำเทคโนโลยีมาประยุกต์ใช้เพื่อเป็นช่องทางในการประชาสัมพันธ์ร้านค้า นำเสนอแคตตาล็อกสินค้า และให้ข้อมูลการลงทุนแฟรนไชส์บนโลกออนไลน์

---

## 1. ความเป็นมา (Background)

ร้าน **วงษ์หิรัญค้าส่ง โคราช** เป็นร้านค้าส่งเบ็ดเตล็ดอันดับ 1 ในจังหวัดนครราชสีมา ที่มีสินค้ามากกว่า 1,000 รายการ ทั้งของเล่นเด็ก เครื่องเขียน ของใช้ในบ้าน เครื่องครัว กีฬา และกิ๊ฟช็อป นอกจากจะเป็นแหล่งรวมสินค้าสำหรับพ่อค้าแม่ค้าแล้ว ทางร้านยังมีบริการส่งเสริมอาชีพด้วย "แพ็กเกจเปิดร้าน 20 บาท" สำหรับผู้ที่ต้องการเริ่มต้นธุรกิจส่วนตัว

ในปัจจุบัน การค้นหาข้อมูลร้านค้าและสินค้ามักเริ่มต้นผ่านทาง Search Engine (Google) หรือการแชร์ลิงก์ทาง Social Media ทางร้านจึงเล็งเห็นความจำเป็นในการสร้าง **ช่องทางติดต่อและนำเสนอสินค้าที่เป็นทางการบนเว็บไซต์** เพื่อสร้างความน่าเชื่อถือ รองรับการทำ SEO ให้ลูกค้าค้นหาเจอได้ง่าย และลดภาระของพนักงานในการตอบคำถามพื้นฐาน เช่น สาขาตั้งอยู่ที่ไหน มีสินค้าประเภทใดบ้าง หรือรายละเอียดการลงทุนเปิดร้านเป็นอย่างไร

ด้วยเหตุนี้ โครงการพัฒนาเว็บไซต์ **WongHiran Website** จึงถูกจัดทำขึ้นด้วยเทคโนโลยี React รูปแบบใหม่ที่ทันสมัย โหลดเร็ว และแสดงผลได้ดีบนมาร์ทโฟน

---

## 2. วัตถุประสงค์ (Objectives)

การพัฒนาระบบเว็บไซต์ทางการ วงษ์หิรัญค้าส่ง มีวัตถุประสงค์หลักดังต่อไปนี้:

1. **เพื่อสร้างความน่าเชื่อถือและภาพลักษณ์ที่ดี:** เป็นศูนย์กลางข้อมูลอย่างเป็นทางการของธุรกิจ นำเสนอความยิ่งใหญ่และสาขาทั้งหมดของวงษ์หิรัญค้าส่ง
2. **เพื่อนำเสนอแคตตาล็อกสินค้า (Product Catalog):** ลูกค้าสามารถเข้าถึงข้อมูล หมวดหมู่ รูปภาพ และราคาสินค้าเบื้องต้นได้ตลอด 24 ชั่วโมง โดยไม่ต้องรอสอบถามผ่านแอดมิน
3. **เพื่อประชาสัมพันธ์แพ็กเกจเปิดร้าน 20 บาท (Franchise Packages):** ให้ข้อมูลละเอียดเกี่ยวกับแพ็กเกจการลงทุน งบประมาณ สิ่งที่ได้รับ และวิธีการเริ่มต้นธุรกิจ เพื่อดึงดูดนักลงทุนรายใหม่
4. **เพื่ออำนวยความสะดวกในการติดต่อและเดินทาง:** รวมแผนที่พิกัด Google Maps ของทั้ง 4 สาขา พร้อมช่องทางลัดสำหรับกดโทรศัพท์หรือแชท LINE ผ่านระบบคลิกปุ่มด่วน (Floating Action Button)
5. **เพื่อขยายฐานลูกค้าผ่านช่องทาง Search Engine (SEO):** วางโครงสร้าง Metadata และ HTML ให้รองรับการติดหน้าแรกบนการค้นหาของ Google สำหรับคีย์เวิร์ดต่างๆ เช่น "ค้าส่ง โคราช", "ร้าน 20 บาท นครราชสีมา"

## 3. ขอบเขตของระบบ (Scope)

ในโครงการระยะนี้ (Phase 1) ตัวเว็บไซต์ถูกพัฒนาในรูปแบบ **Front-End Application** ที่เน้นการแสดงผลเนื้อหาเป็นหลัก (Information & Showcase) 

### 3.1 ขอบเขตสำหรับผู้ใช้งานทั่วไป (Customer Scope)
ผู้ใช้งานคือบุคคลทั่วไป และผู้ประกอบการที่สนใจเข้ามาเยี่ยมชมเว็บไซต์:
- **ข้อมูลหน้าแรก (Home):** ไฮไลต์โปรโมชั่น สินค้าแนะนำ วีดีโอส่งเสริมการขาย หรือบริการหลัก
- **รายการสินค้า (Products):** เรียกดูแคตตาล็อกสินค้า แยกตามหมวดหมู่ พร้อมฟังก์ชันค้นหาสินค้า
- **แพ็กเกจธุรกิจ (Packages):** เรียกดูรายละเอียดของแพ็กเกจเริ่มต้นเพื่อเปิดร้าน 20 บาท แบ่งตามระดับงบประมาณ 
- **เกี่ยวกับเรา (About Us):** ประวัติความเป็นมาของร้าน วิสัยทัศน์ และนโยบาย
- **การติดต่อและสาขา (Contact):** หน้าแสดงข้อมูลติดต่อ เวลาทำการ และแผนที่ Interactive (Google Maps) นำทางสู้สาขาทั้งหมดรวมไปถึงสาขาหลักบ้านเกาะ
- **ขั้นตอนสั่งซื้อ (How to Order):** ให้รายละเอียดเบื้องต้นสำหรับลูกค้าที่ต้องการสั่งซื้อผ่านช่องทางต่างๆ

### 3.2 ขอบเขตของส่วนสนับสนุนระบบ (Technical Scope)
- **การจำลองข้อมูลสินค้า (Mock Data):** ในเวอร์ชันแรก ข้อมูลทั้งหมดจะถูกดึงจากระบบความจำของ Frontend (ไฟล์ `mockData.ts`) เพื่อเร่งความเร็วในการพัฒนาและทดสอบดีไซน์
- **การประมวลผล Responsive:** รองรับการปรับสภาพหน้าจอทุกขนาด ตั้งแต่ Mobile (สมาร์ทโฟน), Tablet, จนถึง Desktop (คอมพิวเตอร์)
- **การดักจับพฤติกรรม (Tracking):** เชื่อมโยงกับระบบ Google Analytics 4 (GA4) เพื่อวิเคราะห์ยอดผู้เข้าชม

### 3.3 ขอบเขตของระบบที่ไม่ได้ครอบคลุม (Out of Scope)
เว็บไซต์นี้ **ยังไม่ครอบคลุม** ถึงระบบหลังบ้านเต็มรูปแบบ ดังต่อไปนี้:
- ระบบผู้ดูแลระบบ (Admin CMS/Dashboard) สำหรับจัดการเนื้อหา หรืออัปโหลดรูปภาพสินค้าแบบ Real-Time 
- ระบบการสั่งซื้อ ชำระเงิน และตะกร้าสินค้าแบบเชื่อมฐานข้อมูล (Checkout/Payment Gateway) ซึ่งจะใช้ระบบ LINE Commerce แทน
- ระบบล็อกอินหรือสมาชิกสำหรับลูกค้า (User Account System)
