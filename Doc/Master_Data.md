# ข้อมูลหลัก (Master Data)

เอกสารนี้รวบรวมข้อมูลอ้างอิงพื้นฐาน (Master Data / Lookup Data) ที่ใช้งานในระบบ WongHiran Website 
ในปัจจุบันระบบอ้างอิงการใช้ข้อมูลสาขา และ หมวดหมู่สินค้า เป็นหลักเพื่อทำระบบฟิลเตอร์ และ สร้างส่วนแสดงผลบนแผนที่

## 1. ข้อมูลสาขาของวงษ์หิรัญ (Branch Data Locations)

ข้อมูลนี้ถูกระบุเข้าไปในระบบผ่าน Component ของ Google Map ซึ่งมีทั้งหมด 4 สาขาทั่วโคราช

| รหัสสาขา | ข้อมูลชื่อสาขา | พิกัดทางภูมิศาสตร์ (Lat, Lng) | โทรศัพท์ |
| :--- | :--- | :--- | :--- |
| `branch-main` | **วงษ์หิรัญค้าส่ง (สาขาใหญ่ บ้านเกาะ)** | 14.9961601, 102.1311029 | 093-502-2828 |
| `branch-sub1` | สาขาย่อย (หัวทะเล) | 14.9620172, 102.1287959 | - |
| `branch-sub2` | สาขาย่อย (จอหอ) | 15.0210087, 102.1332822 | - |
| `branch-sub3` | สาขาย่อย (เซฟวัน) | 14.9545465, 102.0435165 | - |

*(พิกัดด้านบนสามารถเปลี่ยนแปลงได้ในหน้าโค้ดของ `mockData.ts` ตามตำแหน่งที่แม่นยำที่สุด)*

## 2. หมวดหมู่สินค้าตัวอย่าง (Product Category Master)

ใช้สำหรับหน้า **รายการสินค้า (Products)** เผื่อให้ผู้ใช้กดฟิลเตอร์ (Filter Buttons) เพื่อเลือกชมหมวดหมู่ที่ตนสนใจได้ 

| ไอดี (ID) | ชื่อหมวดหมู่ (Name Label) | ไอคอน | หมวดสินค้าอ้างอิง (ตัวอย่าง) |
| :--- | :--- | :---: | :--- |
| `all` | ทั้งหมด | - | นำไปกรองให้แสดงสินค้าทุกอย่างบนหน้าแพลตฟอร์ม |
| `toys` | ของเล่นเด็ก | 🧸 | รถของเล่นเด็ก สไลเดอร์ หุ่นกระบอก |
| `stationery`| เครื่องเขียน | ✏️ | ปากกา ดินสอ แฟ้มเอกสาร ชั้นวางจดหมาย |
| `home` | ของใช้ในบ้าน | 🏠 | กล่องใส่ของ พรมปูพื้น ชั้นพลาสติก |
| `kitchen` | เครื่องครัว | 🍜 | ชาม กะละมัง ถาดเสิร์ฟ จานชามเมลามีน |
| `sports` | อุปกรณ์กีฬา | ⚽ | ลูกฟุตบอล ลูกบาส แร็กเกต ไม้แบตฯ |
| `accessories`| เบ็ดเตล็ด | 🎀 | ยางรัดผม กระจกพกพา หวี สินค้ากิ๊ฟช้อปจิปาถะ |
